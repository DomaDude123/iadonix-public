Self Hosting & Tinkering

I’ve always been interested in self-hosting and homelab work. Much of my tinkering uses an old computer running headless Debian Linux as a small Homeserver/NAS.

  • Hosted a local Bitwarden instance for password management
  • Configured Pi-hole as local DNS along with Unbound as a recursive resolver
  • Hosted a Minecraft server accessible only to VPN-connected users

Skills: Linux, Bash, Python, Networking, Git, Docker, Windows, Security
